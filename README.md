# latex-scientific-resume

Install Latex and `make`

```
sudo apt-get update
sudo apt-get install make texlive-full
```

Latex template for a one-page, two-columns, scientific resume

In your LaTeX editor, build the resume using XeLaTeX

or just run 

```
make
```
